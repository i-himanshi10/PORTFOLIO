Project Overview for BLUESTAR: Industrial Process Monitoring and Control System

DURATION: Sept 2023 - Dec 2023

Software Utilized: Spandan SCADA

Screen Design: Seven screens tailored for 12 stations, managing 13 parameters for precise monitoring and control.

Equipment Monitored: Critical data from field sensors in AC and refrigeration systems.

Data Collection: Efficient data collection via Siemens PLC using MODBUS communication protocol.

Data Storage: Locally hosted SQLite database for lightweight and efficient data storage.

Real-time Communication: Seamless communication between SCADA system, PLC, and other components.

Documentation: Comprehensive PowerPoint presentation for transparency and clarity.

At Bluestar, we take pride in our commitment to excellence, leveraging state-of-the-art technology for unparalleled quality assurance.
